**Mission:** This project aims to support the reduction of mental health cases among youth in Kenya by predicting depression levels using machine learning. The problem addressed is the lack of early detection of mental health risks among students and young adults. By analyzing lifestyle, academic, and social factors, the model identifies patterns associated with depression. This enables early intervention and informed decision-making to improve youth mental well-being.

## Dataset
* **Name:** Student Depression Dataset
* **Source:** Kaggle
* **Author:** Shodolamu Opeyemi
